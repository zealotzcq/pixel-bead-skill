# pixel-bead-skill
opencode skill for pixel bead

Usage：
1. git clone https://github.com/opencodedev/pixel-bead-skill.git

2. copy the folder pixel-bead to your skill folder

3. open opencode/claude code/etc， spell keyword "pixel bead"

4. webpage will be opened as shown below:

![webpage](/screenshot.png)

5. try out the webpage to generate your own pixel bead artwork.

6. click "stop server" to close the skill.


Supported OS:
- Windows 10
- Windows 11

Future Support:
- MacOS
- Ubbuntu
