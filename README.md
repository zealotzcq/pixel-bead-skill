# 生成拼豆图技能

用于在opencode里用技能生成拼豆图

使用方法：
1. git clone https://github.com/zealotzcq/pixel-bead-skill.git

2. 将pixel-bead文件夹复制到你的技能文件夹中

3. 打开opencode/claude code等工具，输入关键词"拼豆"

4. 将打开如下图所示网页：

![webpage](/screenshot.png)

5. 自己玩吧

6. 点击"停止服务" 可以停止技能服务并关闭网页

支持的操作系统：
- Windows 10
- Windows 11

未来将支持（有空再说）：
- MacOS
- Ubuntu
